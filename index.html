About RelationMap

This tool helps visualize complex human relationships. The grouping feature can be used to show the community/organisation each person belongs to.
For the connections:
Dotted line: knowing/recognising each other & had/have interactions;
Solid line: very close relationship;
Dash line: not knowing each other / almost no interaction.
Of course you can use this mind map making tool for other purposes.
Your map will be stored temporarily in your own browser. So, don’t worry about losing your map when closing the browser (unless you delete the cache). Export it as a JSON file to save your work into your device.
